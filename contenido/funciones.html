Funciones en Python - Las 3 más utilizadas

Aprende las funciones básicas, con parámetros y con retorno. ¡Todo lo que necesitas para empezar!

1
Función Simple
Solo ejecuta código

La función más básica. Solo ejecuta un bloque de código cuando la llamas.

Como: Un botón que siempre hace lo mismo.

2
Función con Parámetros
Recibe datos para trabajar

Recibe información (parámetros) y la usa para hacer su trabajo.

Como: Una calculadora que necesita números.

3
Función con Retorno
Devuelve un resultado

Hace un cálculo o proceso y devuelve el resultado.

Como: Un horno que cocina y te da la comida.

1️⃣ Función Simple (sin parámetros ni retorno)

💡 Piénsalo así:

Una función simple es como un botón de "Buenos días" que siempre dice lo mismo cuando lo presionas.

Característica: Siempre hace exactamente lo mismo.

¿Cómo se define?

Estructura básica
def nombre_funcion():
    # Código que se ejecuta
    print("Hola desde la función")

# Para usar (llamar) la función:
nombre_funcion() # Esto ejecuta la función

Ejemplos prácticos

Ejemplo 1: Saludo fijo
# Definir la función
def saludar():
    print("¡Hola! Bienvenido al curso de Python")
    print("Espero que aprendas mucho")
    print("-" * 30)

# Usar la función (puedes llamarla muchas veces)
saludar()
saludar()
saludar()
Ejemplo 2: Mostrar menú
def mostrar_menu():
    print("\n" + "═" * 30)
    print(" MENÚ PRINCIPAL")
    print("═" * 30)
    print("1. Ver perfil")
    print("2. Configuración")
    print("3. Salir")
    print("═" * 30)

# Programa principal
mostrar_menu()
print("Usuario seleccionó opción 1")
mostrar_menu()

🎯 ¿Cuándo usar funciones simples?

✅ Cuando necesitas repetir el mismo código muchas veces

✅ Para organizar tu programa en partes

✅ Cuando el código siempre hace lo mismo (sin variaciones)

2️⃣ Función con Parámetros

💡 Piénsalo así:

Una función con parámetros es como una calculadora: necesitas darle números para que haga la operación.

Característica: Recibe información y trabaja con ella.

¿Qué son los parámetros?

Son variables especiales que reciben datos cuando llamas a la función. Son como "huecos" que se llenan con información.

Sintaxis con parámetros
def saludar_persona(nombre):
    # 'nombre' es el parámetro
    print(f"¡Hola {nombre}! ¿Cómo estás?")

# Llamar la función con diferentes datos
saludar_persona("Ana") # nombre = "Ana"
saludar_persona("Carlos") # nombre = "Carlos"
saludar_persona("María") # nombre = "María"

Ejemplos con múltiples parámetros

Ejemplo 1: Calculadora básica
def sumar(a, b):
    resultado = a + b
    print(f"{a} + {b} = {resultado}")

def multiplicar(x, y):
    resultado = x * y
    print(f"{x} × {y} = {resultado}")

# Usar las funciones
sumar(5, 3) # a=5, b=3
sumar(10, 20) # a=10, b=20
multiplicar(4, 7) # x=4, y=7
Ejemplo 2: Formatear información
def mostrar_usuario(nombre, edad, ciudad):
    print("\n" + "─" * 30)
    print(f"👤 Nombre: {nombre}")
    print(f"🎂 Edad: {edad} años")
    print(f"🏙️ Ciudad: {ciudad}")
    print("─" * 30)

# Llamar con diferentes datos
mostrar_usuario("Ana", 25, "Quito")
mostrar_usuario("Carlos", 30, "Guayaquil")
mostrar_usuario("María", 22, "Cuenca")

📌 Parámetro obligatorio

Debes pasar siempre un valor

def funcion(param):

📌 Parámetro por defecto

Tiene un valor predefinido

def funcion(param=10):

📌 Múltiples parámetros

Puedes tener varios

def func(a, b, c):

3️⃣ Función con Retorno (return)

💡 Piénsalo así:

Una función con retorno es como un horno: le das ingredientes (parámetros) y te devuelve un pastel cocinado (resultado).

Característica: Procesa datos y devuelve un resultado que puedes usar.

¿Qué hace return?

La palabra return toma un valor y lo "devuelve" al lugar donde fue llamada la función. ¡Este valor se puede guardar en una variable!

Sintaxis con return
def duplicar(numero):
    resultado = numero * 2
    return resultado # Devuelve el valor

# Guardar el resultado en una variable
mi_resultado = duplicar(5)
print("El doble de 5 es:", mi_resultado) # 10

# Usar directamente el resultado
print("El doble de 8 es:", duplicar(8)) # 16

Ejemplos prácticos con retorno

Ejemplo 1: Calculadora con retorno
def calcular_area_rectangulo(base, altura):
    area = base * altura
    return area

def calcular_iva(precio):
    iva = precio * 0.12
    return iva

# Usar los resultados
area_sala = calcular_area_rectangulo(5, 3)
iva_producto = calcular_iva(100)

print(f"Área de la sala: {area_sala} m²")
print(f"IVA de $100: ${iva_producto:.2f}")
print(f"Total con IVA: ${100 + iva_producto:.2f}")
Ejemplo 2: Validación con retorno booleano
def es_par(numero):
    if numero % 2 == 0:
        return True
    else:
        return False

def es_adulto(edad):
    return edad >= 18 # Devuelve True o False directamente

# Usar en condiciones
numero = 10
if es_par(numero):
    print(f"{numero} es par")
else:
    print(f"{numero} es impar")

edad = 20
if es_adulto(edad):
    print(f"Con {edad} años eres adulto")

⚠️ ¡IMPORTANTE!

¿Cuál es la diferencia entre print() y return?

print() → Muestra información en pantalla

return → Devuelve un valor para usar en el programa

Una función puede tener ambos, pero normalmente tiene return si necesita devolver un resultado.

Ejercicio 1: Conversor de temperatura

Crea una función que convierta grados Celsius a Fahrenheit usando la fórmula: F = (C × 9/5) + 32

def celsius_a_fahrenheit(celsius):
    fahrenheit = (celsius * 9/5) + 32
    return fahrenheit

# Probar la función
temp_c = 25
temp_f = celsius_a_fahrenheit(temp_c)
print(f"{temp_c}°C = {temp_f:.1f}°F")

# Probemos con 0°C (punto de congelación)
print(f"0°C = {celsius_a_fahrenheit(0):.1f}°F")
print(f"100°C = {celsius_a_fahrenheit(100):.1f}°F")

Ejercicio 2: Calculadora de IMC

Crea una función que calcule el Índice de Masa Corporal (IMC) con la fórmula: IMC = peso / (altura²)

def calcular_imc(peso, altura):
    imc = peso / (altura ** 2)
    return imc

def clasificar_imc(imc):
    if imc < 18.5:
        return "Bajo peso"
    elif imc < 25:
        return "Peso normal"
    elif imc < 30:
        return "Sobrepeso"
    else:
        return "Obesidad"

# Ejemplo de uso
mi_imc = calcular_imc(70, 1.75)
clasificacion = clasificar_imc(mi_imc)
print(f"IMC: {mi_imc:.1f} - Clasificación: {clasificacion}")

Ejercicio 3: Generador de contraseñas simple

Crea una función que genere una contraseña con el patrón: "ABC-123-XYZ" donde ABC son las primeras 3 letras del nombre y XYZ los últimos 3 números del año.

def generar_contraseña(nombre, año):
    # Obtener primeras 3 letras en mayúsculas
    iniciales = nombre[:3].upper()
    
    # Obtener últimos 3 dígitos del año
    digitos = str(año)[-3:]
    
    # Generar número aleatorio entre 100 y 999
    import random
    numero = random.randint(100, 999)
    
    # Formar la contraseña
    contraseña = f"{iniciales}-{numero}-{digitos}"
    return contraseña

# Probar la función
mi_contraseña = generar_contraseña("Fernando", 2024)
print(f"Contraseña generada: {mi_contraseña}")

🎯 Resumen: Las 3 funciones más utilizadas

1️⃣ Función Simple

Para: Ejecutar código repetitivo

No tiene: Parámetros ni retorno

Ejemplo: mostrar_menu()

2️⃣ Función con Parámetros

Para: Trabajar con datos variables

Tiene: Parámetros de entrada

Ejemplo: saludar(nombre)

3️⃣ Función con Retorno

Para: Calcular y devolver resultados

Tiene: Parámetros y return

Ejemplo: calcular_imc(peso, altura)

💡 Consejo final:

Empieza con funciones simples → Añade parámetros cuando necesites datos → Usa return cuando necesites resultados
