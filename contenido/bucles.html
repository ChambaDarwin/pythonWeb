Bucles en Python - Teoría Sencilla

Aprende los conceptos básicos de bucles de manera fácil y práctica. ¡Sin complicaciones!

📚 ¿Qué es un Bucle?

💡 Piénsalo así...

Un bucle es como una playlist que se repite. Cuando termina la última canción, vuelve a empezar desde la primera.

O como cuando comes galletas: "Mientras haya galletas en el plato, sigue comiendo" 🍪

¿Para qué sirven los bucles?

Los bucles nos ayudan a repetir tareas sin tener que escribir el mismo código una y otra vez.

📝 Ejemplo de la vida real:

1 Sin bucle: "Lava plato 1", "Lava plato 2", "Lava plato 3"...
2 Con bucle: "Mientras haya platos sucios, lava un plato"

Tipos de bucles en Python

🔄 Bucle FOR

"PARA cada elemento en una lista, haz algo"

  • Sabemos cuántas veces se repetirá
  • Se usa con listas, rangos, textos
  • Para recorrer elementos

⏱️ Bucle WHILE

"MIENTRAS sea verdad, haz algo"

  • No sabemos cuántas veces se repetirá
  • Se usa con condiciones
  • Para repetir hasta que algo cambie

🎯 En resumen:

FOR = "Haz esto 10 veces" (sabemos el número)

WHILE = "Sigue haciendo esto hasta que dejes de tener hambre" (no sabemos cuándo terminará)

🔄 Bucle FOR - Recorriendo Elementos

💡 Analogía simple:

Un bucle FOR es como revisar cada estudiante en una lista de clase:

"Para cada estudiante en la lista, llama su nombre"

Sintaxis básica

Estructura del bucle FOR
# Para cada elemento en una lista
for elemento in lista:
    # Haz algo con el elemento
    print(elemento)

Ejemplos prácticos

Ejemplo 1: Recorrer una lista
# Lista de frutas
frutas = ["manzana", "banana", "naranja"]

for fruta in frutas:
    print("Me gusta la", fruta)

# Salida:
# Me gusta la manzana
# Me gusta la banana
# Me gusta la naranja
Ejemplo 2: Usar range() para contar
# Contar del 1 al 5
for numero in range(1, 6):
    print("Número:", numero)

# Contar de 2 en 2
for i in range(0, 10, 2):
    print(i) # 0, 2, 4, 6, 8
Ejemplo 3: Recorrer un texto
palabra = "Python"

# Recorrer cada letra
for letra in palabra:
    print("Letra:", letra)

# Salida:
# Letra: P
# Letra: y
# Letra: t
# Letra: h
# Letra: o
# Letra: n

📝 Reglas del bucle FOR:

1. Necesita una lista o algo que recorrer

2. Usa la palabra in (en)

3. Se repite una vez por cada elemento

4. Termina cuando acaba la lista

⏱️ Bucle WHILE - Mientras sea verdad

💡 Analogía simple:

Un bucle WHILE es como esperar el autobús:

"MIENTRAS no llegue el autobús, sigue esperando"

Sintaxis básica

Estructura del bucle WHILE
while condicion:
    # Haz esto mientras la condición sea True
    print("Esto se repite")
    # IMPORTANTE: Cambiar algo para que termine

⚠️ ¡CUIDADO CON LOS BUCLES INFINITOS!

❌ Ejemplo de bucle INFINITO (malo):

# ¡ESTO NUNCA TERMINA!
while True:
    print("Ayuda, estoy atrapado en un bucle")
# Esto imprimirá para siempre 😱

Ejemplos correctos

Ejemplo 1: Contador con while
# Inicializamos el contador
contador = 1

# Mientras contador sea menor o igual a 5
while contador <= 5:
    print("Contador:", contador)
    # Aumentamos el contador (IMPORTANTE)
    contador = contador + 1

print("¡Terminé de contar!")
Ejemplo 2: Validar contraseña
contraseña = ""
intentos = 0

# Mientras no sea la contraseña correcta
while contraseña != "python123" and intentos < 3:
    contraseña = input("Ingresa la contraseña: ")
    intentos = intentos + 1
    
    if contraseña == "python123":
        print("¡Acceso concedido!")
    elif intentos == 3:
        print("Demasiados intentos. Bloqueado.")
Ejemplo 3: Menú interactivo
opcion = ""

while opcion != "4":
    print("\n--- MENÚ ---")
    print("1. Ver saludo")
    print("2. Ver la hora")
    print("3. Contar hasta 3")
    print("4. Salir")

    opcion = input("Elige una opción: ")
    
    if opcion == "1":
        print("¡Hola! ¿Cómo estás?")
    elif opcion == "2":
        print("¡Es hora de aprender Python!")
    elif opcion == "3":
        for i in range(1, 4):
            print(i)

📝 Reglas del bucle WHILE:

1. Necesita una condición que pueda cambiar

2. Se repite mientras la condición sea True

3. SIEMPRE debe haber algo que cambie la condición

4. Termina cuando la condición se hace False

⚡ Control de Bucles - break, continue, pass

💡 Piénsalo como un videojuego:

break = "Salir del nivel" (termina el bucle)

continue = "Saltar este enemigo" (siguiente iteración)

pass = "Esperar un momento" (no hacer nada)

1. break - Terminar el bucle

"ROMPE" el bucle y termínalo inmediatamente

Ejemplo: Buscar un número
numeros = [1, 3, 5, 7, 9, 11]

for num in numeros:
    if num == 7:
        print("¡Encontré el 7!")
        break # Termina aquí
    print("Buscando...", num)

print("Búsqueda terminada")

2. continue - Saltar a la siguiente

"CONTINUA" con la siguiente iteración, saltando el resto

Ejemplo: Saltar números pares
for i in range(1, 11):
    if i % 2 == 0: # Si es par
        continue # Saltar números pares
    print("Número impar:", i)

# Salida: 1, 3, 5, 7, 9 (solo impares)

3. pass - No hacer nada

"PASA" sin hacer nada, es un marcador de posición

Ejemplo: Esqueleto de código
# Cuando no sabes qué poner aún
for i in range(5):
    if i < 3:
        pass # TODO: implementar luego
    else:
        print("Número grande:", i)

🎮 Ejemplo práctico: Juego simple

while True:
    respuesta = input("¿Quieres seguir jugando? (s/n): ")
    
    if respuesta == "n":
        print("¡Gracias por jugar!")
        break # Terminar juego
    elif respuesta == "s":
        print("¡Sigamos jugando!")
        continue # Siguiente ronda
    else:
        pass # Respuesta inválida, preguntar otra vez

📝 Resumen de control:

break = "¡Terminar ahora!" 🛑

continue = "Saltar esto, siguiente por favor" ⏭️

pass = "Aquí va algo, pero todavía no" 🤔

🧩 Ejercicios para practicar

Ejercicio 1: Contar vocales

Escribe un programa que cuente cuántas vocales tiene una palabra.

palabra = "programacion"
vocales = 0

for letra in palabra:
    if letra in "aeiou":
        vocales = vocales + 1

print(f"La palabra '{palabra}' tiene {vocales} vocales")

Ejercicio 2: Tabla de multiplicar

Pide un número al usuario y muestra su tabla de multiplicar del 1 al 10.

numero = int(input("¿Qué tabla quieres ver? "))

print(f"\nTabla del {numero}:")
for i in range(1, 11):
    resultado = numero * i
    print(f"{numero} x {i} = {resultado}")

Ejercicio 3: Adivina el número

Crea un juego donde el usuario deba adivinar un número secreto.

import random

secreto = random.randint(1, 10)
intentos = 0
adivinado = False

while not adivinado and intentos < 5:
    intento = int(input("Adivina el número (1-10): "))
    intentos += 1

    if intento == secreto:
        print(f"¡Correcto! Lo lograste en {intentos} intentos")
        adivinado = True
    elif intento < secreto:
        print("Más alto")
    else:
        print("Más bajo")

if not adivinado:
    print(f"¡Se acabaron los intentos! El número era {secreto}")
