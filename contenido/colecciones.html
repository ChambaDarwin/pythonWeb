Diferencias clave

📋 Lista

  • Se crea con corchetes [ ]
  • Es mutable — puedes cambiarla
  • Permite agregar, eliminar y modificar
  • Ideal para datos que cambian

📦 Tupla

  • Se crea con paréntesis ( )
  • Es inmutable — no se puede cambiar
  • Solo permite leer sus elementos
  • Ideal para datos fijos y constantes
📋

Listas

Fundamento

¿Qué es una lista?

Una lista es una colección ordenada y mutable de elementos. Puede contener datos de distinto tipo: números, texto, booleanos, incluso otras listas.

python 
# Dos formas de crear una lista vacía
lista = []
lista = list()

# Lista con elementos
frutas = ["manzana", "pera", "uva"]
numeros = [1, 2, 3, 4, 5]
Método

Agregar elementos — append()

El método append() agrega un elemento al final de la lista. Se usa cuando no te importa la posición.

python 
lista = []
lista.append(2)
lista.append(6)
lista.append(7)
print(lista)
Resultado
[2, 6, 7]
Técnica

Llenar con bucle for

Combinar for + range() + input() permite pedirle al usuario que llene la lista con sus propios valores.

python 
lista = []
for i in range(3):
    n = int(input(f"Ingresa un número en la posición {i+1}: "))
    lista.append(n)
print(lista)
Resultado (ejemplo)
[2, 4, 5]
Método

Insertar en posición — insert()

insert(posición, valor) agrega un elemento en una posición específica. Los demás elementos se desplazan a la derecha.

💡 Los índices empiezan en 0. El primer elemento es el índice 0, el segundo es el índice 1, y así sucesivamente.
python 
lista = [3, 4, 5]
lista.insert(1, 200)  # inserta 200 en el índice 1
print(lista)
Resultado
[3, 200, 4, 5]
Actualizar

Modificar un elemento

Para cambiar un valor, accede a él por su índice y asígnale el nuevo valor con =.

python 
lista = [3, 4, 5]
lista[0] = 500  # cambia el primer elemento
print(lista)
Resultado
[500, 4, 5]
Eliminar

Eliminar por valor — remove()

remove(valor) busca y elimina la primera aparición del valor indicado.

python 
lista = [3, 4, 5, 4, 4]
lista.remove(4)  # elimina solo el primer 4
print(lista)
Resultado
[3, 5, 4, 4]
Eliminar

Eliminar por posición — pop()

pop() elimina y devuelve el elemento en la posición indicada. Sin argumento, elimina el último.

python 
lista = [3, 4, 5, 4, 4]

lista.pop()       # sin argumento → elimina el último
print(lista)    # [3, 4, 5, 4]

lista.pop(1)     # con índice → elimina el elemento en pos 1
print(lista)    # [3, 5, 4]

Resumen de métodos

append(x)
Agrega x al final de la lista
insert(i, x)
Inserta x en la posición i
remove(x)
Elimina la primera aparición de x
pop(i)
Elimina y devuelve el elemento en i
len(lista)
Devuelve la cantidad de elementos
sort()
Ordena la lista de menor a mayor

Tuplas

Fundamento

¿Qué es una tupla?

Una tupla es una colección ordenada e inmutable. Una vez creada, sus elementos no pueden cambiar. Se usa para datos que no deben modificarse, como coordenadas, fechas o configuraciones.

python 
tupla = (4, 5, 6)
print(tupla)      # (4, 5, 6)
print(tupla[2])   # 6  → accede por índice
Técnica

Recorrer una tupla con for

Igual que las listas, puedes iterar sobre una tupla con un bucle for.

python 
tupla = (4, 5, 6)
for i in tupla:
    print(i)
Resultado
4
5
6
Avanzado

Desempaquetar tuplas

Puedes asignar cada elemento de una tupla a una variable diferente en una sola línea. Es muy útil cuando una función devuelve múltiples valores.

python 
def operacion(a, b):
    suma           = a + b
    resta          = a - b
    multiplicacion = a * b
    return suma, resta, multiplicacion   # devuelve tupla

# Desempaquetamos los 3 valores en 3 variables
suma, resta, multi = operacion(7, 3)
print(f"Suma: {suma}, Resta: {resta}, Multi: {multi}")
Resultado
Suma: 10, Resta: 4, Multi: 21
Métodos

Métodos disponibles

Las tuplas tienen solo dos métodos nativos porque son inmutables.

python 
tupla = (1, 2, 2, 3, 2)
print(tupla.count(2))   # 3  → cuántas veces aparece el 2

tupla2 = (5, 6, 7, 8, 3)
print(tupla2.index(8))  # 3  → posición del 8
Conversión

Convertir entre lista y tupla

Usa tuple() para convertir una lista en tupla, y list() para convertir una tupla en lista.

python 
lista = [2, 3, 5, 6]

# Lista → Tupla
tupla = tuple(lista)
print(tupla)    # (2, 3, 5, 6)

# Tupla → Lista
lista2 = list(tupla)
print(lista2)   # [2, 3, 5, 6]

Métodos de tupla

count(x)
Cuenta cuántas veces aparece x
index(x)
Devuelve la posición de x
len(tupla)
Devuelve la cantidad de elementos
tuple(lista)
Convierte una lista en tupla
