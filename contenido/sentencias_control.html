if y match en Python

Aprende cuándo usar sentencias condicionales tradicionales (if/elif/else) y cuándo usar la nueva estructura match (Python 3.10+)

if/elif/else (Python 2.x - 3.x)
match/case (Python 3.10+)

🎯 Sentencia if

La sentencia if es la estructura condicional tradicional en Python, disponible desde las primeras versiones.

Sintaxis Básica

Estructura if/elif/else
# Evaluación de condiciones
edad = 25

if edad < 13:
    print("Niño")
elif edad < 20:
    print("Adolescente")
elif edad < 65:
    print("Adulto")
else:
    print("Adulto mayor")

# Con operadores lógicos
if edad >= 18 and edad <= 30:
    print("Joven adulto")

Características Principales

✅ Flexibilidad total

Puedes usar cualquier expresión booleana como condición.

✅ Anidamiento ilimitado

Puedes anidar if dentro de otros if según sea necesario.

✅ Compatibilidad universal

Funciona en todas las versiones de Python 2.x y 3.x.

🔀 Sentencia match 3.10+

La sentencia match (introducida en Python 3.10) permite coincidencia de patrones más expresiva.

Sintaxis Básica

Estructura match/case
# Coincidencia de patrones simple
dia = "lunes"

match dia:
    case "lunes" | "martes" | "miércoles" |
         "jueves" | "viernes":
        print("Día laboral")
    case "sábado" | "domingo":
        print("Fin de semana")
    case _: # Caso por defecto
        print("Día inválido")

# Coincidencia con asignación
match usuario:
    case {"nombre": nombre, "edad": edad}:
        print(f"Usuario {nombre}, {edad} años")

Características Principales

✅ Coincidencia de patrones

Puede coincidir con patrones complejos, no solo valores simples.

✅ Sintaxis más limpia

Para muchos casos, es más legible que múltiples if/elif.

✅ Asignación en coincidencia

Puede extraer valores mientras hace coincidencias.

📊 Comparación Directa

🤔 ¿Cuál usar? Guía de Decisión

✅ Usa if/elif/else cuando...

  • 📌 Necesitas compatibilidad con Python < 3.10
  • 📌 Las condiciones son expresiones booleanas complejas
  • 📌 Necesitas evaluar rangos numéricos específicos
  • 📌 Requieres anidamiento profundo de condiciones
  • 📌 Las condiciones no son solo de igualdad
  • 📌 Evalúas múltiples variables independientes

✅ Usa match/case cuando...

  • 🎯 Tienes Python 3.10+ y es aceptable como requisito
  • 🎯 Estás comparando una variable contra múltiples valores
  • 🎯 Necesitas coincidencia de patrones complejos
  • 🎯 Quieres extraer valores durante la coincidencia
  • 🎯 Buscas máxima legibilidad para casos específicos
  • 🎯 Trabajas con estructuras de datos anidadas

🚀 Ejemplos Avanzados

🎯 if Avanzado

Condiciones complejas
# Validación múltiple de rangos
temperatura = 25
humedad = 70
velocidad_viento = 15

if (20 <= temperatura <= 30 and
    humedad < 80 and
    velocidad_viento < 20):
    print("Día perfecto para salir")
elif temperatura > 35 or humedad > 90:
    print("Mejor quedarse en casa")

🔀 match Avanzado

Patrones complejos
# Coincidencia con estructuras
respuesta = {
    "status": 200,
    "data": {"user": "Juan", "id": 123}
}

match respuesta:
    case {"status": 200, "data": datos}:
        print(f"Éxito: {datos}")
    case {"status": 404}:
        print("Recurso no encontrado")
    case {"status": s} if s >= 500:
        print("Error del servidor")
    case _:
        print("Respuesta desconocida")

🎯 Conclusión

if/elif/else sigue siendo esencial para condiciones complejas y cuando necesitas compatibilidad con versiones anteriores.

match/case es una excelente adición para Python 3.10+, especialmente cuando trabajas con coincidencia de patrones y quieres mejorar la legibilidad del código.

Recomendación: Usa match cuando sea apropiado para mejorar la claridad, pero mantén if en tu arsenal para situaciones más complejas o cuando necesites compatibilidad.
