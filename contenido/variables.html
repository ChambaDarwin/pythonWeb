¿Qué es una variable?

Una variable es como una CAJA donde guardas cosas. Tiene:

  • 📌 NOMBRE (para identificarla)
  • 🎁 CONTENIDO (lo que guarda dentro)
  • 🏷️ ETIQUETA (qué tipo de cosa es)

En Python, creas una variable simplemente dándole un nombre y poniendo algo dentro.

1️⃣ ¿Cómo declarar una variable?

💡 Piénsalo así:

Declarar una variable es como etiquetar una caja vacía y luego ponerle algo dentro.

En Python es súper fácil: nombre = valor

Forma de crear variables
# Así se crea una variable en Python:
nombre = "Ana"
edad = 25
altura = 1.65
es_estudiante = True

# Python entiende automáticamente el tipo
# No necesitas decir "esto es texto" o "esto es número"

El símbolo "=" no es igualdad matemática

El = asigna, no compara
# En programación, = significa "GUARDA ESTO"
x = 10 # Guarda 10 en la variable x
x = 20 # Ahora guarda 20 (cambia el valor)

# Para comparar (ver si son iguales) se usa ==
if x == 20: # ¿x es igual a 20?
    print("Sí, x vale 20")

2️⃣ Reglas para nombrar variables

📝 ¡IMPORTANTE!

Los nombres de variables tienen reglas, como los nombres de personas:

✅ Pueden tener: letras, números (pero no empezar con número), guión bajo _

❌ NO pueden tener: espacios, símbolos especiales (excepto _), palabras reservadas

Reglas de nomenclatura

✅ CORRECTO

edad = 25

nombre_usuario = "Ana"

precio_total = 99.99

_temporal = "temp"

❌ INCORRECTO

1nombre = "Ana" (empieza con número)

nombre usuario = "Ana" (tiene espacio)

precio-total = 99.99 (tiene guión)

if = 10 (palabra reservada)

Convenciones (buenas prácticas)

Estilos recomendados
# snake_case (recomendado en Python)
nombre_completo = "Ana García"
edad_usuario = 25
es_estudiante_activo = True

# Nombres descriptivos (dicen qué guardan)
precio = 19.99 # BIEN: dice qué es
x = 19.99 # MAL: no dice nada

# Constantes (valores que no cambian, en MAYÚSCULAS)
PI = 3.1416
IVA = 0.12

3️⃣ Tipos de variables más comunes

🎁 Tipos de cajas:

Así como hay cajas para libros, cajas para ropa y cajas para comida, las variables también tienen tipos.

Python detecta automáticamente el tipo según lo que guardes.

Texto (str)

Para palabras y frases

"Hola", 'Python'

Enteros (int)

Para números sin decimal

25, -10, 1000

Decimales (float)

Para números con punto

3.14, -2.5, 99.99

Booleanos (bool)

Para verdadero/falso

True, False

Ejemplos de cada tipo

Variables de diferentes tipos
# 1. Texto (string) - se usa con comillas
nombre = "Carlos"
mensaje = 'Hola mundo'
direccion = """Calle Principal Número 123""" # Texto de varias líneas

# 2. Números enteros (int)
edad = 30
cantidad_productos = 15
temperatura = -5 # Puede ser negativo

# 3. Números decimales (float)
precio = 19.99
promedio = 8.75
pi = 3.1415926535

# 4. Booleanos (verdadero/falso)
es_mayor = True
tiene_licencia = False
esta_activo = True

Verificar el tipo de una variable

Función type()

# ¡No necesitas declararlo!

4️⃣ Operaciones con variables

➕ Operaciones matemáticas:

Puedes hacer operaciones con variables como haces con números normales.

¡Y también puedes modificar su contenido después!

Modificar variables

Cambiar el valor de una variable
# 1. Crear una variable
contador = 0
print("Contador:", contador) # 0

# 2. Cambiar su valor
contador = contador + 1 # Ahora vale 1
print("Contador:", contador) # 1

# 3. Forma abreviada para sumar
contador += 1 # Igual que: contador = contador + 1
print("Contador:", contador) # 2

# También funciona con otros operadores
precio = 100
precio *= 1.12 # Aumenta 12% (IVA)
print("Precio con IVA:", precio) # 112.0

Operaciones entre variables

Usar variables juntas
# Crear variables
precio_producto = 25.50
cantidad = 3
descuento = 5.00

# Calcular total
subtotal = precio_producto * cantidad
total = subtotal - descuento

print("Subtotal:", subtotal) # 76.5
print("Descuento:", descuento) # 5.0
print("Total a pagar:", total) # 71.5

# Concatenar texto (unir variables de texto)
nombre = "Ana"
apellido = "García"
nombre_completo = nombre + " " + apellido
print(nombre_completo) # Ana García
