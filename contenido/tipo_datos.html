Tipos de Datos Básicos en Python

Aprende los 4 tipos de datos fundamentales que necesitas para empezar a programar

str Texto
int Enteros
float Decimales

ABC
String (str)
Texto y palabras

¿Para qué sirve?

Para representar texto: nombres, mensajes, direcciones, etc.

Características:

  • Va entre comillas: "hola" o 'hola'
  • Puede contener letras, números y símbolos
  • Se puede cortar, unir y modificar
  • Python lo detecta automáticamente
123
Integer (int)
Números enteros

¿Para qué sirve?

Para contar cosas: edad, cantidad, posiciones, etc.

Características:

  • No tiene punto decimal
  • Puede ser positivo o negativo
  • Se puede sumar, restar, multiplicar
  • Ideal para bucles y contadores
3.14
Float
Números decimales

¿Para qué sirve?

Para medidas precisas: precios, temperaturas, porcentajes.

Características:

  • Tiene punto decimal
  • Puede ser positivo o negativo
  • Útil para cálculos científicos
  • Precisión limitada (15 decimales)
✓✗
Boolean (bool)
Verdadero o Falso

¿Para qué sirve?

Para decisiones y condiciones: ¿está activo? ¿es mayor de edad?

Características:

  • Solo dos valores: True o False
  • Mayúscula inicial (True, False)
  • Resultado de comparaciones
  • Fundamental para if/else

📝 Ejemplos Prácticos

1. Strings (Texto)

Ejemplos de strings
# Strings con comillas simples o dobles
nombre = "Ana"
apellido = 'García'
direccion = "Calle Principal 123"

# String de varias líneas
mensaje = """Hola, este es un mensaje de varias líneas"""

# Operaciones con strings
nombre_completo = nombre + " " + apellido
print(nombre_completo) # Ana García
print("Hola " * 3) # Hola Hola Hola

2. Integers (Enteros)

Ejemplos de enteros
# Números enteros positivos y negativos
edad = 25
cantidad_productos = 100
temperatura = -5
puntuacion = 0

# Operaciones matemáticas
suma = 10 + 5 # 15
resta = 20 - 8 # 12
multiplicacion = 6 * 7 # 42
division_entera = 10 // 3 # 3 (sin decimales)

3. Floats (Decimales)

Ejemplos de decimales
# Números con punto decimal
precio = 19.99
pi = 3.141592
temperatura = 23.5
porcentaje = 0.12 # 12%

# Operaciones con decimales
total = 10.5 + 3.2 # 13.7
iva = 100 * 0.12 # 12.0
division = 10 / 3 # 3.3333333333333335

# Redondeo
redondeado = round(pi, 2) # 3.14

4. Booleans (Verdadero/Falso)

Ejemplos de booleanos
# Valores booleanos directos
es_mayor = True
tiene_licencia = False
esta_activo = True

# Booleanos de comparaciones
resultado1 = 10 > 5 # True
resultado2 = 10 == 5 # False
resultado3 = "Ana" == "ana" # False (diferente mayúscula)

# Operadores lógicos
and_result = True and False # False
or_result = True or False # True
not_result = not True # False

🔄 Conversiones entre tipos

A veces necesitas cambiar un tipo a otro:

Funciones de conversión
# De texto a número (si es posible)
texto_numero = "25"
entero = int(texto_numero) # 25 (int)
decimal = float(texto_numero) # 25.0 (float)

# De número a texto (siempre funciona)
edad = 30
edad_texto = str(edad) # "30" (str)

# A booleano (0, "", None son False, lo demás True)
bool1 = bool(0) # False
bool2 = bool(1) # True
bool3 = bool("") # False (string vacío)
bool4 = bool("Hola") # True

str()

Convierte a texto

str(25) → "25"

int()

Convierte a entero

int("25") → 25

float()

Convierte a decimal

float("3.14") → 3.14

bool()

Convierte a True/False

bool(1) → True
