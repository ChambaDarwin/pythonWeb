import trae el módulo tkinter al proyecto. El alias as tk le pone un apodo corto para no escribir "tkinter" cada vez que lo uses.

Definir la función principal
def crea un bloque de instrucciones con nombre. Esta función se ejecutará cada vez que el usuario haga clic en el botón. Por ahora solo se define, no se ejecuta.
Como escribir una receta de cocina: la anotas en el cuaderno ahora, pero la cocinas solo cuando llegue el momento (cuando el usuario haga clic).

Leer el campo de texto
.get() extrae como texto (string) lo que el usuario escribió en la caja de entrada. Se guarda en la variable number.

Como el portero de un club que solo deja pasar si traes entrada; si llegaste con las manos vacías, no entras.

Convertir texto → número entero
.get() siempre devuelve texto (ej. "42"). int() lo convierte al número 42 para poder compararlo matemáticamente con > y <.
Es como cambiar un billete de papel por monedas: mismo valor, pero ahora puedes contarlo y compararlo.

Primera condición: ¿es positivo?

Segunda condición: ¿es negativo?
elif = "else if". Solo se evalúa si la condición anterior (>0) fue falsa.
Si no ganaste el primer premio de la rifa, revisas si ganaste el segundo.

Si no fue positivo ni negativo, matemáticamente solo puede ser cero. El else captura ese caso sin necesidad de escribir == 0.
Si no subiste ni bajaste de piso en el ascensor, sigues exactamente en la planta baja.

Crear un contenedor (Frame)
Un Frame es una caja invisible con fondo amarillo que agrupa y organiza los widgets hijos. bg = background (color de fondo).
Una bandeja de cocina: pones todos los ingredientes ordenados dentro antes de llevarlos a la mesa.

Inflar un globo dentro de una caja: se expande hasta ocupar todo el espacio disponible.

El cartel de "Tome un número" en un banco: le indica al usuario exactamente qué debe hacer.
22

Campo de texto editable (Entry)
Crea la caja donde el usuario escribe el número. Se guarda en name para poder leerla después con name.get().
El formulario en blanco esperando a que escribas tu nombre.

name.pack()

24

Botón de acción
Crea el botón con el texto "Saludar". command=parImpar vincula el evento clic con la función parImpar(). Nota: se pasa el nombre sin paréntesis (referencia, no llamada).
El timbre de una puerta: cuando alguien lo presiona, dispara una acción específica (el mayordomo corre a abrir).

El corazón que late sin parar: mientras late, la aplicación sigue viva y respondiendo.

Conceptos Clave

Los bloques de construcción detrás de ambas apps, con analogías del mundo real.

import … as tk

Importar trae herramientas prefabricadas. El alias as tk le pone un apodo corto para no repetir el nombre completo.

def función():

Define instrucciones con nombre. Solo se ejecutan cuando alguien las llama. Es como escribir una receta: la anotas ahora, la usas después.

int(valor)

Convierte texto a número entero. "42" → 42. Sin esto no puedes hacer comparaciones matemáticas.

if / elif / else

Bifurcaciones: el programa toma solo un camino. Como un árbol de decisiones.

tk.Tk() + mainloop()

Tk() construye la ventana. mainloop() la mantiene viva. Sin mainloop, se cerraría en milisegundos.

tk.Frame()

Contenedor que agrupa widgets. Como una bandeja que lleva varios objetos juntos y organizados.

tk.Entry() + .get()

Campo de texto editable. .get() lee su contenido cuando lo necesitas.

Button (command=fn)

command apunta a la función que se ejecuta al presionar. El timbre que activa al mayordomo.

Label + .config()

Etiqueta de texto. .config(text=...) la actualiza en tiempo real sin recrearla.

.pack(fill, expand, pady)

Gestor de geometría automático. fill="both" estira en X e Y. pady agrega margen vertical.

Concatenación (+)

"hola " + nombre + " como estas" une cadenas. Como pegar pedazos de papel para formar una oración.

Validación != ""

Verificar que el campo no esté vacío antes de procesar. Como revisar que el vaso tenga agua antes de beberlo.
