Entrada de Datos en Python

Aprende a capturar información del usuario con input()

📌 Función input()

La función input() es la forma más básica y común de capturar datos del usuario por teclado en Python.

nombre = input("Ingresa tu nombre: ")
print("Hola,", nombre)

💡 Nota importante

La función input() siempre devuelve una cadena de texto (string), incluso si el usuario ingresa números.

🎯 Tipos de Entrada

1. Entrada de Texto

# Entrada básica de texto
nombre = input("¿Cuál es tu nombre? ")
ciudad = input("¿En qué ciudad vives? ")
print(f"Hola {nombre} de {ciudad}!")

2. Entrada de Números

# Para números, debemos convertir el string a int o float
edad_str = input("Ingresa tu edad: ")
edad = int(edad_str) # Convertir a entero

# O en una sola línea:
edad = int(input("Ingresa tu edad: "))

# Para números decimales:
altura = float(input("Ingresa tu altura (metros): "))

⚠️ Cuidado con las conversiones

Si el usuario ingresa algo que no sea un número válido, int() o float() lanzarán un error. Para manejar esto, puedes usar try-except.

🔧 Ejemplos Prácticos

Calculadora Simple

num1 = float(input("Primer número: "))
num2 = float(input("Segundo número: "))
print(f"Suma: {num1 + num2}")

Formulario Personal

nombre = input("Nombre: ")
edad = int(input("Edad: "))
email = input("Email: ")
print(f"Usuario: {nombre}, {edad} años, contacto: {email}")

Ejemplo: Programa de Bienvenida

# Programa que saluda al usuario
print("=== BIENVENIDO ===")
nombre = input("¿Cómo te llamas? ")
edad = int(input("¿Cuántos años tienes? "))
ciudad = input("¿De dónde eres? ")

print(f"\n¡Hola {nombre}!")
print(f"Tienes {edad} años y vives en {ciudad}.")
print(f"El año que viene tendrás {edad + 1} años.")

⚡ Mejores Prácticas

✅ Validación de Entrada

Siempre valida los datos del usuario, especialmente cuando necesitas números:

while True:
    try:
        edad = int(input("Ingresa tu edad: "))
        break # Salir del bucle si la conversión es exitosa
    except ValueError:
        print("Error: Debes ingresar un número válido.")

✅ Mensajes Claros

Siempre muestra mensajes claros al usuario sobre qué tipo de dato esperas:

# Buen ejemplo:
input("Ingresa tu peso en kg (ej: 65.5): ")

# Mal ejemplo:
input("Peso: ")

✅ Manejo de Múltiples Valores

Para capturar múltiples valores en una sola línea:

# Separados por espacio
datos = input("Ingresa nombre y edad: ").split()
nombre = datos[0]
edad = int(datos[1])

# O separados por coma
x, y = input("Ingresa X e Y separados por coma: ").split(",")
