Operadores en Python

Aprende los 4 tipos de operadores fundamentales para realizar operaciones en Python

+ - * / Aritméticos
> < == Comparación
and or not Lógicos
= += -= Asignación
+
Aritméticos
Para cálculos matemáticos
Operador Nombre Ejemplo
+ Suma 5 + 3 → 8
- Resta 10 - 4 → 6
* Multiplicación 3 * 4 → 12
/ División 10 / 3 → 3.333
// División entera 10 // 3 → 3
% Módulo (resto) 10 % 3 → 1
** Potencia 2 ** 3 → 8
=
Comparación
Para comparar valores
Operador Nombre Ejemplo
== Igual a 5 == 5 → True
!= Diferente de 5 != 3 → True
> Mayor que 10 > 5 → True
< Menor que 3 < 7 → True
>= Mayor o igual 5 >= 5 → True
<= Menor o igual 4 <= 4 → True
&
Lógicos
Para combinar condiciones
Operador Nombre Ejemplo
and Y lógico True and False → False
or O lógico True or False → True
not NO lógico not True → False
=
Asignación
Para asignar valores
Operador Nombre Ejemplo
= Asignación x = 5
+= Suma y asigna x += 3 (x = x + 3)
-= Resta y asigna x -= 2 (x = x - 2)
*= Multiplica y asigna x *= 2 (x = x * 2)
/= Divide y asigna x /= 2 (x = x / 2)

📝 Ejemplos Prácticos de Operadores

1. Operadores Aritméticos

Ejemplos de operaciones matemáticas
# Operaciones básicas a = 10 b = 3 suma = a + b # 13 resta = a - b # 7 multi = a * b # 30 division = a / b # 3.333... entera = a // b # 3 (sin decimales) resto = a % b # 1 (10 ÷ 3 = 3 sobra 1) potencia = a ** b # 1000 (10³) print(f"Suma: {suma}") print(f"Resta: {resta}") print(f"Multiplicación: {multi}") print(f"División: {division:.2f}") # .2f para 2 decimales print(f"División entera: {entera}") print(f"Resto: {resto}") print(f"Potencia: {potencia}")

2. Operadores de Comparación

Ejemplos de comparaciones
# Comparaciones numéricas edad = 18 nota = 85 es_mayor = edad >= 18 # True es_menor = edad < 18 # False es_aprobado = nota >= 70 # True es_excelente = nota > 90 # False es_igual = edad == 18 # True es_diferente = edad != 21 # True # Comparaciones de strings nombre1 = "Ana" nombre2 = "ana" son_iguales = nombre1 == nombre2 # False (Python distingue mayúsculas) son_iguales_ignore_case = nombre1.lower() == nombre2.lower() # True print(f"¿Es mayor de edad? {es_mayor}") print(f"¿Está aprobado? {es_aprobado}") print(f"¿Los nombres son iguales? {son_iguales}")

3. Operadores Lógicos

Ejemplos de lógica booleana
# Ejemplos prácticos con operadores lógicos tiene_licencia = True tiene_seguro = False edad = 25 saldo = 1000 # Puede conducir si tiene licencia Y es mayor de edad puede_conducir = tiene_licencia and edad >= 18 # True # Puede viajar si tiene licencia O tiene seguro puede_viajar = tiene_licencia or tiene_seguro # True # No puede pagar si NO tiene suficiente saldo puede_pagar = not (saldo < 500) # True (1000 NO es menor que 500) # Combinación compleja puede_comprar_carro = (edad >= 18 and tiene_licencia) and (saldo > 20000 or tiene_seguro) print(f"¿Puede conducir? {puede_conducir}") print(f"¿Puede viajar? {puede_viajar}") print(f"¿Puede pagar? {puede_pagar}") print(f"¿Puede comprar carro? {puede_comprar_carro}")

4. Operadores de Asignación

Ejemplos de asignaciones
# Asignación básica x = 10 print(f"Valor inicial de x: {x}") # 10 # Operadores compuestos (más eficientes) x += 5 # Equivalente a: x = x + 5 print(f"x += 5 → {x}") # 15 x -= 3 # Equivalente a: x = x - 3 print(f"x -= 3 → {x}") # 12 x *= 2 # Equivalente a: x = x * 2 print(f"x *= 2 → {x}") # 24 x /= 4 # Equivalente a: x = x / 4 print(f"x /= 4 → {x}") # 6.0 # Ejemplo práctico: Contador contador = 0 contador += 1 # Incrementar en 1 contador += 1 # Incrementar en 1 contador -= 1 # Decrementar en 1 print(f"Valor final del contador: {contador}") # 1

🎯 Resumen: Los 4 Tipos de Operadores

➕ Aritméticos

Para: Cálculos matemáticos

Clave: +, -, *, /, //, %, **

Ejemplo: 10 + 5 * 2

⚖️ Comparación

Para: Comparar valores

Clave: ==, !=, >, <, >=, <=

Ejemplo: edad >= 18

🔀 Lógicos

Para: Combinar condiciones

Clave: and, or, not

Ejemplo: edad >= 18 and tiene_licencia

📝 Asignación

Para: Asignar valores

Clave: =, +=, -=, *=, /=

Ejemplo: x += 5

💡 Reglas Importantes

1. = asigna, == compara

2. and requiere ambos True

3. or requiere al menos uno True

4. not invierte el valor
